# TravelMind AI Agent - Frontend

A world-class, hackathon-winning frontend for the TravelMind AI Agent platform.

## 🚀 Features

- **AI-Powered Travel Planning**: Intelligent multi-agent system for personalized itineraries
- **Stunning UI/UX**: Glassmorphism design with Framer Motion animations
- **Full Authentication**: JWT-based auth with protected routes
- **Interactive Chat**: Real-time AI conversation with reasoning panel
- **Trip Management**: Save, view, edit, and delete travel plans
- **PDF Export**: Download itineraries as professional PDFs
- **Analytics Dashboard**: Beautiful charts and statistics
- **Trip Comparison**: Compare multiple trips side by side
- **Responsive Design**: Works perfectly on all devices

## 🛠️ Tech Stack

- **React 19** - Latest React with modern features
- **Vite** - Lightning fast build tool
- **TailwindCSS** - Utility-first CSS framework
- **Framer Motion** - Smooth animations
- **React Router v6** - Client-side routing
- **Axios** - HTTP client
- **Recharts** - Data visualization
- **React Hot Toast** - Beautiful notifications
- **React Icons** - Icon library
